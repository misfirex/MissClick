# MissClick

**MissClick** is a small desktop app that keeps track of missed clicks, click streaks, and rage-worthy mouse moments.

It sits on your screen as a simple floating pill-style counter and gives quick visual feedback when you miss clicks.

Built for people who are gaming, testing aim, messing about with mouse accuracy, or just want a stupidly satisfying little click tracker.

---

## Features

- Floating Windows-style pill display
- Tracks missed clicks
- Shows miss streaks
- Play / pause button
- Colour switch option
- Rage Quit button
- Lightweight desktop app
- Simple and easy to use

---

## Controls

| Key / Button | Action |
|---|---|
| `P` | Play / pause |
| `C` | Change colour |
| `Rage Quit` | Close the app |

---

## What It Does

MissClick runs as a small desktop overlay.

When active, it tracks your mouse click behaviour and updates the counter on screen. The app is designed to be simple, funny, and lightweight rather than packed with unnecessary menus.

Future versions may include:

- Sound effects for miss streaks
- Custom messages
- More colour themes
- Different display styles
- Installer version
- Auto-updates

## Installation

### Option 1: Download the app

1. Go to the **Releases** section of this GitHub page.
2. Download the latest `.exe` file.
3. Run the app.

Windows may show a warning because this is an unsigned app from an independent developer. This does not automatically mean the app is unsafe.

---

## Important Windows Warning

Because MissClick is not yet code-signed, Windows SmartScreen or Google may sometimes flag the file as unknown or potentially unsafe.

This usually happens with new independent apps that do not yet have a trusted publisher certificate.

The app does **not** intentionally collect personal data, passwords, browser history, or private files.
